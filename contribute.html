Let's Build Awesome, Together

Brackets, an open source code editor built with JS, HTML and CSS, has been designed from the ground up to make it easy for web developers, like you, to hack on and extent its functionality.

Try It

Brackets is still in the oven and needs to bake a little longer before it's a full-featured code editor. The milestone builds available on download.brackets.io are there for contributors and those special kind of crazy designers and developers who like to live on the edge.

Performance, stability and features are all in flux. Use at your own risk. However, if you like living on the bleeding edge, just view the wiki for info on how to download and run Brackets.
