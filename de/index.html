Warum Brackets nutzen?

Im Browser designen

Brackets ist Ihrem kreativen Arbeiten nicht im Weg. Visuelle Werkzeuge werden im Editor eingeblendet, während HTML und CSS beim Programmieren an den Browser gesendet wird.

Ein erhabenes Erlebnis

Brackets ist nicht nur schön, sondern auch mächtig. Das Schreiben von Code wird Ihnen in Brackets Spaß machen. Brackets ist Ihnen nicht im Weg und bietet nur genau so viel Hilfe, wie Sie brauchen.

Gemacht mit ♥ und JavaScript

Brackets ist ein Open-Source-Projekt, mit einer aktiven und wachsenden Entwickler-Gemeinschaft – gemacht von Web-Entwicklern wie Ihnen!

Kontinuierlich weiterentwickelt

Brackets ist eine Spielwiese zum Ausprobieren neuer Ideen für Web-Werkzeuge. Neue Funktionen und Erweiterungen werden alle 2,5 Wochen veröffentlicht. So erhalten Sie das ganze Jahr über Geschenke.

Neue Funktionen

Live HTML-Entwicklung

Bei der Vorschau eines Projekts mit Brackets werden alle CSS- und HTML-Änderungen beim Tippen aktualisiert.

CSS-Code-Intelligenz für SASS

Beim Coden von .SCSS-Dateien stellt Brackets jetzt Code-Intelligenz für CSS zur Verfügung. Selbst nagelneue Fähigkeiten wie die CSS-Regionen flow-into und flow-from werden unterstützt.

Neue CSS-Regel beim Schnell-Bearbeiten

Verwenden Sie die Schaltfläche Neue Regel im CSS-Schnell-Editor (Strg+E/⌘E), um eine Regel für das aktuell markierte Tag bzw. die Klasse/ID zu erstellen, selbst wenn es keine bestehenden Regeln gibt. Sie können den Selektor nach der Erstellung verändern.

Schnell-Editor für Animations-Beschleunigung

Wenn sich die Auswahl innerhalb einer Timing-Funktion für eine CSS-Animation befindet, können Sie den Schnell-Editor (Strg+E/⌘E) nutzen, um die Zeitkurve visuell zu bearbeiten.
